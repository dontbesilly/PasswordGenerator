# Password generator with cli

* -h - for help
* -v - version
* -o - options
  * --len - length of password
  * --type - 1: only `letters`; 2: with `numbers`; 3: with symbols `!@#$%^&*()`

System variables -> Path -> Edit -> New -> Path to .exe  
Use: `pgt -o --len 10 --type 3`  
Default(length 8, type 2): `pgt`
